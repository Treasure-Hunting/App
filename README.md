## Requirements
- Python >= 3.5
- Django >= 2.2.4

## Installation
```console
$ git clone https://github.com/Treasure-Hunting/App.git && cd App
$ python manage.py migrate
$ python manage.py loaddata initial_data
$ python manage.py runserver
```
